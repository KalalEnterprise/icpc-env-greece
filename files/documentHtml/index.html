GRCPC Environment Guide

VS Code

  • How to open VS Code:
    To open VS Code you need to right click on the desktop or folder you want to open and select the "Open in Terminal". In the terminal type "code" and press enter.
  • How to enable/disable extensions:
    To enable/disable the extensions you select the four boxes on the sidebar of the vs code (or press Ctrl+Shift+X) and navigate to the Installed tab. There are all the Installed extensions. After that you select the Installed extension and choose to enable it or dsable it.
